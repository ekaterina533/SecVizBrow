# SecVizBrowser
Этот проект представляет собой веб-приложение, предназначенное для визуализации и систематизации научных статей. Основной целью проекта является создание удобного, интерактивного интерфейса для поиска, фильтрации и управления научными публикациями.
# Основной функционал системы:
- главная страница со списком статей, функцией поиска и фильтрации по категориям и тегам;
- система авторизации с разграничением прав (администратор/пользователь);
- админ-панель для управления содержимым сборника:
  - добавление, редактирование и удаление статей;
  - загрузка изображений;
  - импорт статей из BibTeX-файлов;
  - управление тегами и категориями;
- динамическая фильтрация статей по категориям.
# Технические особенности:
- Frontend реализован с использованием HTML, CSS и JavaScript, макеты подготовлены в Figma;
- Backend построен на Python-фреймворке Flask с использованием SQLAlchemy для работы с базой данных;
- для хранения данных используется реляционная база MySQL;
- реализована система миграций с помощью Flask-Migrate;
- взаимодействие с пользователем и защита данных реализованы с помощью Flask-WTF, Flask-Login, werkzeug и других библиотек.
Проект разделён на клиентскую и серверную части с чёткой архитектурой и использованием Blueprint-модулей во Flask
